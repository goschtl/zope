megrok.genshi
=============

megrok.genshi makes it possible to use Genshi templates in Grok. 

For more information on Grok and Genshi see:

- http://grok.zope.org/
- http://genshi.edgewall.org/

Requirements
------------

- Genshi. Tested with v 0.4.4.
- Grok v0.11 or later. Tested with 0.11.
