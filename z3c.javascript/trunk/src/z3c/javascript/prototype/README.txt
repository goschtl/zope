==========================================
 Prototype Javascript library as Resource
==========================================

This package registers the prototype javascript library as a browser
resource called `prototype.js`.

Additionally if zc.resourcelibrary is installed it also registers a
resourcelibrary called 'z3c.javascript.prototype'.

See: http://prototype.conio.net/

Note: This package is always kept in sync with the script.aculo.us
library requirements which is provided by the
``z3c.javascript.scriptacoulus`` package
