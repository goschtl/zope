Installing CMFDecor

  1. Unpack the tarball.

  2. Copy or link the 'CMFDecor' directory into the 'Products'
     directory of your INSTANCE_HOME / SOFTWARE_HOME.

  3. Restart Zope.

  4. Log into the Zope management interface and navigate to the
     root of the CMF Site you want to use with CMFDecor.

  5. Instantiate an External Method with Module name CMFDecor.Install
     and function name install

  6. Run the external method via the "Test" tab in its management 
     interface
