========
Zope3org
========

This package contains the subpackages of the Zope3org project that
started at the NeckarSprint.

wikification
------------

A package that translates arbitrary static html documents into a wikified
version that allows the user to easily browse and edit the document tree.

comment
-------

A simple comment implementation that stores comment as file like data in the
annotations of an object.

importer
-----------

An import mechanism for translating http://www.zope.org/Wikis/DevSite into
the new Zope3 format.

kupusupport
-----------

Bindings and adapters that are necessary to use the Kupu WYSIWYG editor
within Zope3 in general and the wikification project in particular.
