Thanks for installing Zope!

To access the Zope Management Interface, point your favorite web browser
at http://localhost:8080/manage.  When prompted for a username and
password, provide the username 'admin' and the password you provided
during the installation process.

Once you've logged in to the Zope Management Interface, press the "Go"
button next to the "Zope Quick Start" option in the top frame.  This
link provides information on how to quickly get started using Zope.
