============
CMF.buildout
============
-------------------------
Build CMF 2.1 + Zope 2.10
-------------------------

Introduction
============

Builds CMF 2.1 with Zope 2.10 from develop eggs located in ``src``.

Dependencies
============

Requires Python 2.4

Usage
=====
::

  $ python2.4 bootstrap/bootstrap.py
  $ ./bin/buildout
  $ ./bin/test
  $ ./bin/instance
