3.3 Configuring and Running a ZEO Server

Edit /usr/local/etc/zeo.conf appropriately for your desired setup. This configuration file controls the port on which ZEO will listen for connections, the user and group IDs under which the server will be executed, and the location of the concrete Storage object that will be made network-accessible.
