3.2.2 Running a server

The start.py script in the ZEO directory can be used to start a server. Run it with the -h option to see the various values. If you're just experimenting, a good choise is to use python ZEO/start.py -D -U /tmp/zeosocket to run ZEO in debug mode and with a Unix domain socket.
