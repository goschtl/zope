3.2.2 Installation

Installing the ZEO package is easy. Just run python setup.py install. This will install the ZEO/ package into your Python installation, and copy various files into their proper locations: zeo.conf will be put into /usr/local/etc/, a zeo startup script will be put in /etc/rc.d/init.d/, and the zeod daemon program will be placed in /usr/local/bin.
